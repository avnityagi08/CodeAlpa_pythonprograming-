# Hangman Game

A simple text-based Hangman game built using Python.

## Description
This project is a console-based Hangman game where the player guesses a hidden word one letter at a time.

## Features
- Random word selection
- 5 predefined words
- Maximum 6 incorrect guesses
- Input validation
- Win and Lose conditions

## Concepts Used
- Python
- Random Module
- While Loop
- If-Else Statements
- Strings
- Lists

## How to Run

1. Make sure Python is installed.
2. Open a terminal in the project folder.
3. Run:

```bash
python hangman.py
```
